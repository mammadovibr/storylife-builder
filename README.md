# StoryLife Builder

Visual editor for interactive text stories with AI-assisted branching, desktop Electron support, and an installable offline iPad version.

## Open the app

**[Launch StoryLife Builder](https://mammadovibr.github.io/storylife-builder/)**

For iPad:

1. Open the link in Safari while online.
2. Wait until the toolbar says **Offline ready**.
3. Tap **Share -> Add to Home Screen**.
4. Open the home-screen app once before travelling.

The installed app can then start without an internet connection. JSON projects can be loaded through the iPad Files picker.

## Development

```powershell
npm install
npm run dev
npm test
npm run build
npm run build:ipad
```

The published PWA is built from the `docs` directory. Core source code, tests, Electron integration, AI story-planning pipeline, and build scripts are included in this repository.
