# StoryLife Builder on iPad

1. Upload the repository with the generated `docs` folder to GitHub.
2. In GitHub open **Settings -> Pages** and publish the `main` branch from `/docs`.
3. Open the GitHub Pages address in Safari while online.
4. Wait until the top status says **Offline ready**.
5. In Safari tap **Share -> Add to Home Screen**.
6. Open StoryLife from the new home-screen icon once before the trip.

After publishing a new version, open it online and wait for **Offline ready** again so the iPad can replace its cached copy.
