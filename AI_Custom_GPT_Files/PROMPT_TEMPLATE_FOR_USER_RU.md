# Шаблон запроса для Custom GPT

Скопируй этот текст и вставь в Custom GPT вместе со своей историей.

```text
Сделай из моей истории готовый project.json для StoryLife Builder.

Верни только чистый JSON, без Markdown и без пояснений.

Требования:
- version: 1
- id сцен: scene_1, scene_2, scene_3 и так далее
- id choices: choice_1_1, choice_1_2, choice_2_1 и так далее
- связи между сценами только через targetNodeId
- startSceneId должен указывать на первую сцену
- все targetNodeId должны указывать на существующие сцены
- добавь parameters, если в истории есть числовые состояния
- добавь flags, если игре нужно помнить события
- добавь effects к choices, если выбор что-то меняет
- добавь conditions к choices, если выбор зависит от флага или параметра
- добавь conditionalTargets, если один выбор может вести в разные сцены
- imagePath и soundPath оставляй пустыми, если я не дал конкретные файлы
- layoutType подбирай по смыслу сцены
- position расставь так, чтобы дерево было читаемым слева направо

Моя история:

[ВСТАВЬ СЮДА ИСТОРИЮ ОБЫЧНЫМ ЯЗЫКОМ]
```

## Пример короткого запроса

```text
Сделай интерактивную историю на 12 сцен.
Главная героиня София хочет попасть на концерт.
Игрок выбирает: попросить помощи у мамы, поговорить с папой или искать билет самой.
Нужны флаги: поговорила с мамой, помогла папе, нашла билет.
Нужны параметры: уверенность, настроение.
Должно быть 3 концовки: попала на концерт, не попала но сохранила отношения с семьей, испугалась и решила попробовать позже.

Верни только project.json.
```
